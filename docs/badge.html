Badge

A badge is used to add additional information to the content. It is most widely used with icons, and avatars.

Badge on Icons

Badge on Icons are mostly used for notification or showing some useful information like current items in wishlist, unread messages in social media website.

Badge on Avatars

To create avatar with badge simply add badge-container class. Badge on Avatars are used to represent user's current status which can be either online/offline/busy.
