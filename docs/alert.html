Alert

An alert displays a short, important message in a way that attracts user's attention without interrupting the user's task.

Alert with Icons

To create alert with icons simply add alert class with contextual classes to add different colors on top of that. For example, alert-error and a message for notifying some error has occured.

Alert with Icons and outline

To create alert with icons and outline simply add alert-(alert-type)-outline class with contextual classes to add different colors on top of that. For example, alert-error-outline and a message for notifying some error has occured with outline.
