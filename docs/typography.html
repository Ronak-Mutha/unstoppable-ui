Typography

Unstoppable UI provides a lot of text utilities that you can play around with.

Headings

You can use h1, h2, h3, h4, h5, and h6 tags for headings. You can use classes as well i.e. .h1, .h2, .h3, .h4, .h5, and .h6 to imitate similar visual reference but don't want to use semantic heading tags.
